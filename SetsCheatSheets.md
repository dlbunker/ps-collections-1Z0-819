## Set methods to Know

| Method        | Return Type          | Parameters                            |
|---------------|----------------------|---------------------------------------|
|add()          | boolean              | (E element)                           |
|remove()       | boolean              | (Object object)                       |
|isEmpty()      | boolean              | none                                  |
|size()         | int                  | none                                  |
|clear()        | void                 | none                                  |
|contains()     | boolean              | (Object object)                       |
|equals()       | boolean              | (Object object)                       |
|hashCode()     | int                  | none                                  |
